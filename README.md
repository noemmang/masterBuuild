# 🖥️ MasterBuild

## 📌 ¿Qué es MasterBuild?

MasterBuild es un comparador de precios de componentes de PC, similar a Idealo, pero enfocado exclusivamente en hardware.

La aplicación permite a los usuarios buscar componentes, comparar precios entre distintas tiendas y acceder directamente a ellas para realizar la compra. MasterBuild **no vende productos**, actúa como intermediario mostrando la mejor información disponible.

---

## 🎯 Objetivo del proyecto

Este proyecto ha sido desarrollado como Trabajo de Fin de Grado (TFG) del ciclo **Desarrollo de Aplicaciones Web (DAW)**.

El objetivo principal es crear una plataforma completa que ayude a los usuarios a montar su propio PC de forma informada, facilitando:

- Comparación de precios en múltiples tiendas
- Configuración de PCs con validación de compatibilidad
- Comparación de especificaciones entre componentes
- Visualización avanzada (como comparador 3D de gabinetes)
- Seguimiento de precios y alertas

---

## ⚙️ Tecnologías utilizadas

- **Backend:** Laravel 11 (API REST)
- **Frontend:** Angular 18
- **Base de datos:** PostgreSQL
- **Arquitectura:** MVC desacoplado (frontend y backend separados)

---

## 🚀 Funcionalidades principales

- 🔍 Buscador de componentes con filtros avanzados
- 💸 Comparador de precios entre tiendas
- 🧠 Configurador de PC con validación de compatibilidad
- 📊 Comparador de especificaciones
- 🖥️ Comparador visual de gabinetes (3D)
- ⭐ Sistema de guardados
- 🔔 Alertas de precio
- 👤 Gestión de usuario
